Phase 14 of 16 — CAP Framework

SAP Cloud Application Programming Model (CAP)

8 days • Dec 8-15, 2026
What is this? CAP is SAP's opinionated Node.js (or Java) framework for building services on BTP. You declare entities in CDS files; CAP auto-generates OData v4 endpoints, DB deployment, and auth wiring.

Why should you learn this? CAP is the SAP way of building modern services. Understanding CAP = you can write BTP backends, feed Fiori Elements UIs, and hook AI actions — all with less code than pure Node/Express.

Topics covered: cds init, schema.cds (entities/types/associations/compositions), service.cds (expose + restrict), custom handlers (before/on/after), SQLite for local dev, HANA Cloud deploy, mta.yaml multi-target descriptor, cf deploy
